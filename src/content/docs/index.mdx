## What is PDTF 2.0?

The Property Data Trust Framework is an open standard for making property data independently verifiable. Every piece of data — from title registers to EPC certificates — is cryptographically signed by its source, can be verified by anyone, and can be revoked when it's no longer valid.

No need to trust the platform serving the data. Verify the proof.

PDTF 2.0 replaces trust in intermediaries with trust in cryptography. Built on W3C Verifiable Credentials, Decentralised Identifiers, and a federated Trusted Issuer Registry, it provides the infrastructure for a property data ecosystem where data provenance is provable and verification is automatic.

## The entity graph

PDTF 2.0 decomposes the monolithic property data pack into nine distinct entities — Transaction, Property, Title, Person, Organisation, Ownership, Representation, DelegatedConsent, and Offer — each independently verifiable and composable.

This means a title register credential issued by HM Land Registry can be verified independently of the transaction it belongs to. An EPC credential can travel with the property, not just the transaction. And ownership changes can be revoked without invalidating the entire data pack.
